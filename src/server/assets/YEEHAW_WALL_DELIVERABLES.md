## 2) Machine-readable matrix

- **JSON (Variant A):** `src/server/assets/yeehaw-wall-48-a.json` — `{ "width": 48, "height": 48, "pixels": number[][] }`
- **JSON (Variant B):** `src/server/assets/yeehaw-wall-48-b.json` — same format.

Values: `0` = background, `1` = foreground, `2` = shadow (optional; unused here).

## 3) TypeScript export

See `src/server/assets/YeehawWall48.ts`:

- `export const YEEHAW_WALL_48_A: number[][]` — Variant A.
- `export const YEEHAW_WALL_48_B: number[][]` — Variant B.
- `export const YEEHAW_WALL_48: number[][]` — Default (B).

## 4) Integration snippet

Included in `YeehawWall48.ts` as a comment block. Summary:

```ts
const TILE_BG = 0;
const TILE_FG = 15;
const TILE_SHADOW = 14; // optional

import { YEEHAW_WALL_48 } from './YeehawWall48.js';

function renderYeehawMural(wallX: number, wallY: number): void {
  const mural = YEEHAW_WALL_48;
  for (let row = 0; row < mural.length; row++) {
    for (let col = 0; col < mural[row].length; col++) {
      const v = mural[row][col];
      const tileId = v === 0 ? TILE_BG : v === 2 ? TILE_SHADOW : TILE_FG;
      setTile(wallX + col, wallY + row, tileId);
    }
  }
}
renderYeehawMural(0, 0); // anchor top-left
```

## 5) Validation checklist

| Check | Status |
|-------|--------|
| Dimensions exactly 48×48 | Yes |
| Min stroke thickness 2px for lines (lasso, arm, hat brim) | Yes |
| No facial features, no buttons, no 1px hairlines | Yes |
| Silhouette readable at small scale (hat + arm + lasso) | Yes |
| Optional text: only "YEEHAW", blocky, 3–4px stroke, bottom 12–14 rows | Yes (Variant B) |
| 1–2px margin around edges | Yes |
| Two-color (0/1); no shadow used | Yes |
